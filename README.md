# Dipal Thaker Portfolio

A personal portfolio website built to showcase my work, experience, skills, and projects in full-stack development, data analytics, and software engineering.

## About

This portfolio highlights my background as an MS in Information Management student at the University of Illinois Urbana-Champaign, along with my experience across software engineering, cloud, analytics, and product-focused development.

It includes sections for:

- About Me
- Projects
- Experience
- Education
- Skills
- Leadership
- Extracurriculars
- Contact

## Built With

- HTML5
- CSS3
- JavaScript
- Responsive layout
- Custom UI styling
- Dark mode support
- Interactive project cards
- Smooth navigation and transitions

## Features

- Clean and modern single-page portfolio
- Responsive design for different screen sizes
- Custom cursor interactions
- Dark and light theme toggle
- Project showcase with flip-card details
- Experience timeline
- Skills and leadership sections
- Direct links to LinkedIn, GitHub, and resume
